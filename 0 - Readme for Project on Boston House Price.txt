
The dataset for Boston House Price analysed and it was taken from UCI repository.  
Unimodel data visualization was done. Regression algorithms like: Lasso, Elastic Net, CART, SVR and KNN were evaluated.
Linear regression looked good (KNN). Ensemble methods were used like:AdaBoast, Gradient Boosting, Random Forests and Extra Trees
and  to finalize the model by getting most from Gradient Boosting.
